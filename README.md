# 🎙️ Voice AI Pipeline (Multi-language)

## 📌 Overview
This project focuses on building a multi-language AI voice generation pipeline using custom voice cloning.

Languages supported:
- Hindi
- Marathi
- English

The system is designed for scalable batch processing and optimized for efficient GPU usage.

---

## 🚀 Features

- Voice cloning using custom datasets
- Multi-language audio generation
- Batch script processing
- Audio post-processing (WAV formatting, normalization)
- Optimized for low-latency inference

---

## 🧠 Objective

To develop a lightweight and efficient AI voice system aligned with:
- Low compute usage
- Fast inference
- Scalable deployment

This project will be used for experimentation on GPU platforms like RunPod.

---

## ⚙️ Tech Stack

- Python
- AI Voice Models (TTS)
- FFmpeg / Audio Processing
- RunPod GPU Infrastructure

---

## 🔄 Workflow

1. Input script (text)
2. Language selection
3. Voice generation
4. Audio processing
5. Final WAV output

---

## 📦 Future Improvements

- Model compression (Parameter optimization)
- Faster inference pipeline
- API integration
- Automation for bulk content generation
